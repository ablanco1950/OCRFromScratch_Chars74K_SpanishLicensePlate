# OCRFromScratch_Chars74K_SpanishLicensePlate
OCR from scratch using Chars74 Dataset: http://www.ee.surrey.ac.uk/CVSSP/demos/chars74k/ applied to the case of Spanish car license plates   or any other with format NNNNAAA. The hit rate is lower than that achieved by pytesseract: in a test with 21 images, 14 hits are reached  while with pytesseract the hits are 17.
